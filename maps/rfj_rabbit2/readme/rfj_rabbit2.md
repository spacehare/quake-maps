# tools used

- [TrenchBroom 2025.3](https://trenchbroom.github.io/)
- [ericw-tools-2.0.0-alpha10-win64](https://github.com/ericwa/ericw-tools/releases/tag/2.0.0-alpha10)
- [MESS 1.2.3](https://pwitvoet.github.io/mess/)
- [some of my own jank scripts](https://github.com/spacehare/rabbit_quake)

## compile stack

| tool       | unique args                    |
| ---------- | ------------------------------ |
| Export Map |                                |
| MESS       | `-config quake`                |
| pp.py      |                                |
| QBSP       | `-leaktest -litwater -bsp2`    |
| VIS        | `-noambient`                   |
| LIGHT      | `-extra -emissivequality high` |
