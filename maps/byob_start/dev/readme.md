# Bring Your Own Bottle

BYOB is a Quake jam where mappers are required to make their own textures. Uses [Copper 1.35](https://lunaran.com/copper/). A modern sourceport, like IronWail, is recommended.

Co-hosted by Wons and Rabbit, with assistance from Quiet and the Quake Mapping Discord staff. The active mapping period started June 18th, 2026 and ended July 3rd, 2026.

You may not use any textures or assets included in this project for generative AI in any capacity.

Please read the individual readmes too!

# installation instructions

extract the contents of the ZIP inside of your quake sourceport's folder. it should create a `byob/` folder NEXT TO your `id1/` folder.
