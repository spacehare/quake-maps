# Bring Your Own Bottle

BYOB is a Quake jam where mappers are required to make their own textures. Based on [Copper 1.35](https://lunaran.com/copper/).

Co-hosted by Wons and Rabbit, with assistance from Quiet and the Quake Mapping Discord staff. The active mapping period started June 18th, 2026 and ended July 3rd, 2026.

Please read the individual readmes too!

You may not use any textures or assets included in this project for generative AI in any capacity.

A modern sourceport, like IronWail, is recommended.

# installation instructions

extract the contents of the ZIP inside of your quake sourceport's folder. it should create a `byob/` folder NEXT TO your id1 folder, and also be NEXT TO ironwail.exe or quakespasm.exe, etc
